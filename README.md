# Zepto Dataset Cleaning using Python

## Dataset
The dataset was downloaded from Kaggle:
https://www.kaggle.com/datasets/palvinder2006/zepto-inventory-dataset

## Objective
The goal of this project is to clean the Zepto inventory dataset using Python and Pandas.

## Data Cleaning Steps
1. Import dataset using pandas
2. Check dataset structure using info() and describe()
3. Identify and remove missing values
4. Remove duplicate records
5. Convert data types
6. Remove invalid price values
7. Create new column for discount percentage
8. Clean text columns using string functions
9. Export cleaned dataset

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Output
A cleaned dataset ready for analysis and visualization.
